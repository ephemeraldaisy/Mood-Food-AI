# Mood-Food-AI
Repository for Mood Food AI, a Gemini-based food recommendation system based on your mood. 
제목: Mood Food AI 🍱

한 줄 설명: 사용자의 10가지 기분에 맞춰 성균관대(수원) 좌표 기반의 맞춤형 메뉴를 추천하는 지능형 웹 앱입니다.

라이브 데모: 사용자님의 배포 주소

2. 기술 스택 (Tech Stack)
Backend: Streamlit

AI Engine: Gemini 3 Flash (Paid Tier)

Maps: Folium & Streamlit-Folium

Hosting: Streamlit Community Cloud

3. AI 도구 활용 및 인용 (AI Tool Usage & Citation)
제공해주신 소스 코드에서도 강조되었듯, AI를 활용한 프로젝트에서는 인용 가이드라인을 명시하는 것이 표준입니다.

AI 모델: 코드 구조화, 디버깅, 평가 함수 브레인스토밍을 위해 Gemini 3.1 Pro를 활용하였습니다.

인용 예시: "Python script for implementing the engine and agents." Gemini, 2026 version, Google.
